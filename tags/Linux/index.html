Linux下特别权限位-setuid，setgid，sticky

通常，我们用八进制表示一个权限时，如 644 或者 755，省略了最前面的一个特别权限位，完整地表示是 0644 或者 0755，而第一位就是特别权限位。这里着重要说的是三个特别权限位：setuid，setgid 和 sticky 位。
