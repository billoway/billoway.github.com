Icon

命名方法，举例



在 info.plist 使用 key CFBundleIconFiles，注意与 CFBundleIconFile 区分开，CFBundleIconFiles 的优先级高于 CFBundleIconFile 省略扩展名，直接写 Icon，会自动识别。如果要加扩展名，必须把所有的图片名都添加到 CFBundleIconFiles 的 array 中，特别需要注意的是，如果想自动识别，命名必须严格按照标准，且 key 为 string 类型。如果不自动识别，key 为 array 类似并将完整文件名添加进去。
